## 简介
WeBASE-Data-Collect为区块链数据监管服务，主要提供以下功能：

- 查看链上的交易数量和交易详情。
- 通过关键字搜索，查询交易属于哪条链，哪个机构，哪个合约。保证链上数据可查可管。
